## **for문**

**for문은 조건 검사 후 참일경우 for문 내부를 실행하고 다시** 

**돌아와서 조건을 검사한다. 조건이 거짓이 될때까지 반복해준다.**

**ex) 구구단**

**실행**
```Java
for(int i=2;i<=9;i++) {
            for(int j=1;j<=9;j++) {
                System.out.print(i + " * " + j + " = " + i*j + "   ");
            }
            System.out.print("\n");
        }for(int i=2;i<=9;i++) {
            for(int j=1;j<=9;j++) {
                System.out.print(i + " * " + j + " = " + i*j + "   ");
            }
            System.out.print("\n");
        }
```
